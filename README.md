# Ψηφιακό Ρολόι σε Python

Ένα απλό, κομψό και ελαφρύ ψηφιακό ρολόι με Γραφικό Περιβάλλον (GUI), φτιαγμένο εξ ολοκλήρου στην Python. 

##  Τι κάνει αυτό το πρόγραμμα;
* Δείχνει την τρέχουσα ώρα του υπολογιστή σε μορφή **Ώρα:Λεπτά:Δευτερόλεπτα** (π.χ. 14:30:05).
* Ανανεώνεται αυτόματα κάθε 1 δευτερόλεπτο, χωρίς να χρειάζεται να πατήσεις κάποιο κουμπί.
* Έχει εντυπωσιακή εμφάνιση που θυμίζει ψηφιακά ρολόγια "neon" (μαύρο φόντο με έντονα πράσινα γράμματα).
* Το παράθυρο έχει σταθερό μέγεθος για να δείχνει πάντα όμορφο και συμμαζεμένο.

## Τι χρησιμοποιήσαμε;
* **Python 3** (Η βασική γλώσσα προγραμματισμού).
* **`tkinter`**: Η βιβλιοθήκη για να φτιάξουμε το παράθυρο. Εδώ μάθαμε την εντολή `.after()`, η οποία λειτουργεί σαν χρονόμετρο και λέει στο πρόγραμμα να ανανεώνει την οθόνη συνέχεια!
* **`time`**: Η βιβλιοθήκη της Python που διαβάζει τη σωστή ώρα από το σύστημα του υπολογιστή.
